# dist-upgrade

Update Node and npm.

--------

## Installation:

`npm i -g dist-upgrade`

Or use `npx` to avoid installing it globally: `npx dist-upgrade`.

## Usage:

```
dist-upgrade    # update global Node and npm
dist-upgrade -h # show this help
```

## FAQ
