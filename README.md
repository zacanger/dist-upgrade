# dist-upgrade

Update Node and npm.

--------

## Installation:

`npm i -g dist-upgrade`

Or use `npx` to avoid installing it globally: `npx dist-upgrade`.

## Usage:

`dist-upgrade`

## FAQ

* It's not working!
  * Only works on OSs with a `/bin/sh`
* I have that and it's still not working!
  * Please file an issue and I'll take a look!
