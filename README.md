# dist-upgrade

Install latest global Node, npm, and npx

[Donate](https://ko-fi.com/zacanger)

--------

## Usage:

Recommended: `npx dist-upgrade`

Or, if you want to install it globally, `npm i -g dist-upgrade`, then `dist-upgrade`.
